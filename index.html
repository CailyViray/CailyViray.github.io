Glossary

Bit
A binary digit, either 0 or 1.
Byte
A sequence of 8 bits.
Roundoff
Error when bits are not enough to represent numbers with full precision.
Analog Data
Values that change smoothly, like live music volume.
Lossless
Compression that allows full recovery of original data.
Lossy
Compression that discards some data permanently.
Metadata
Data about data, like file details.
Computing Device
A physical device that runs programs.
Computer Network
A group of devices sending/receiving data.
Path
Series of connections between sender and receiver.
Protocol
Agreed rules for communication.
Fault Tolerant
System keeps working even if parts fail.
Scalability
System can expand to meet new demands.
Bandwidth
Max transmission capacity of a system.
URL
Easy-to-remember web address (e.g., www.example.com).
Physical Network
Hardware that carries internet signals.
Internet Protocol (IP)
Assigns unique addresses and routes data.
TCP
Ensures reliable, ordered, error-checked delivery of data.
UDP
Faster, less reliable protocol.
Packets
Small chunks of data messages are split into.
Routers
Forward data across networks via fastest path.
DNS
Translates domain names into IP addresses.
HTTP
Protocol for web pages and media.
WWW
System of linked pages browsable with HTTP.
Redundancy
Multiple pathways ensure reliability.
