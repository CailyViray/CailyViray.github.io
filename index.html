Glossary

Bit
A binary digit, either 0 or 1.
Byte
A sequence of 8 bits.
Roundoff
Error from insufficient bits.
Analog Data
Values that change smoothly.
Lossless
Compression that preserves all data.
Lossy
Compression that discards data.
Metadata
Data about data.
Computing Device
A physical device that runs programs.
Computer Network
Group of devices sharing data.
Path
Series of connections.
Protocol
Rules for communication.
Fault Tolerant
System keeps working despite failures.
Scalability
System can expand or contract.
Bandwidth
Maximum transmission capacity.
URL
Easy-to-remember web address.
Physical Network
Hardware carrying signals.
Internet Protocol (IP)
Assigns addresses and routes data.
TCP
Reliable data delivery protocol.
UDP
Fast but less reliable protocol.
Packets
Small chunks of messages.
Routers
Forward data across networks.
DNS
Translates domain names.
HTTP
Protocol for web pages.
WWW
Linked pages on the web.
Redundancy
Multiple network pathways.

Cybersecurity Threats

Learning about cybersecurity threats is essential because our personal, academic, and professional lives rely heavily on digital systems. Viruses, phishing attacks, and DDoS attacks can lead to data loss, financial damage, or compromised personal information. These threats occur because attackers exploit human error, software vulnerabilities, or weak security measures to gain unauthorized access, steal sensitive information, or disrupt online services. Understanding these risks helps us protect ourselves, build secure systems, and make responsible digital choices.

🐛 Viruses
Programs that harm systems.
📧 Phishing
Tricking people to share info.
🌐 DDoS
Overloading systems to shut them down.
